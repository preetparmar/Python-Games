# T-REX RUSH
The famous Dino Game in `Python` using `PyGame`

## Gameplay
![Game Play](/TRex-Rush/Assets/Images/gameplay.gif)

## Controls
- `Space Bar` to Jump
- `Down` to duck
- If `Down` is pressed while jumping it will descend quicker
- Press `Q` or `Escape` to quit the game anytime
